Detailed Description

Module containing tools for calibrating the SKA-LOW station.

This module contains tools to calibrate the SKA-LOW station and is part of the SKA-low mini project. It contains functions to calculate the spherical wave modes, the EEPs, and to estimate the gains using the SteEFCal algorithm.
